# Det Intelligente Hus
## Krav
#### Componenter

- LCD Display
- RGB Light 
- Keypad
- Portentometer x 2
- Power Supply
- DC Motor
- DHT11
- Servo Motor

#### Funtioner Main

1. Temperaturen og luftfugtigheden skal kunne overvåges i hele huset
2. Små lamper skal kunne skifte farve alt efter om der er varmt eller kold i rummet.
3. Der skal et infoskærm op i stuen så der er mulighed for at se temperaturen i de forskellige rum.
4. Der skal en loftventilator op i stuen som styre sig selv efter temperaturen.
5. Alle vindueren skal ufskiftes med nye smart windows. Så de selv kan åbner hvis nødvendigt.

#### Funtione Optional

1. Hoveddøren skal kunne låses ved hjælp af et keycard.
